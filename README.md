# JavaScript Algorithms

A curated collection of JavaScript algorithms, data structures, coding patterns, and problem-solving solutions implemented using modern JavaScript (ES6+), with a focus on clean, readable, and efficient code

## Topics Covered

- Arrays
- Strings
- Objects
- Recursion
- Searching Algorithms
- Sorting Algorithms
- Hash Maps
- Sets
- Stacks
- Queues
- Linked Lists
- Trees
- Graphs
- Two Pointers
- Sliding Window
- Dynamic Programming
- Greedy Algorithms
- Backtracking
- Bit Manipulation
- Math & Number Problems

## Purpose

This repository aims to:

- Practice Data Structures and Algorithms (DSA)
- Improve problem-solving skills
- Prepare for technical interviews
- Learn optimized approaches and time complexity
- Keep a collection of reusable algorithm implementations

## Technologies

- JavaScript (ES6+)
- Node.js
  
## How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/javascript-algorithms.git
```

Run any JavaScript file:

```bash
node filename.js
```
